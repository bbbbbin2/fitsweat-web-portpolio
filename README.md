# Fitsweat
React와 Spring Boot 기반의 운동 기록 웹사이트   
## 개발 목적
Fit = 체력 + 건강, Sweat = 노력 + 성과 뜻을 가진 Fitsweat  
  
올바른 운동방법을 알지 못하여 어려움을 겪는 사람들을 위해 사용자 친화적인 운동 가이드를 제공하고   
운동 기록을 통해 사용자의 꾸준한 발전 및 체계적으로 운동 기록을 관리 수 있는 웹페이지 기획
